# SSRF Guardian 

A cybersecurity project designed to detect and prevent Server-Side Request Forgery (SSRF) attacks.

##  Features
- Detects malicious URLs and SSRF attack patterns
- Real-time monitoring and analysis
- Dashboard for attack visualization
- Chrome extension for client-side protection

##  Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- Security: API-based validation and filtering

##  How to Run
1. Clone the repository
2. Install dependencies:
   npm install
3. Start backend:
   node server.js
4. Open frontend
